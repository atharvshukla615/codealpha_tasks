# Iris Flower Classification

## Objective

This project classifies Iris flowers into three species:

* Iris Setosa
* Iris Versicolor
* Iris Virginica

## Dataset

The Iris dataset contains 150 flower samples with:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

## Machine Learning Model

Random Forest Classifier

## Results

The model was trained and evaluated using an 80:20 train-test split and achieved high classification accuracy.
