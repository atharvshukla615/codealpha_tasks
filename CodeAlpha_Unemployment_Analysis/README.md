# Unemployment Analysis with Python

## Objective

Analyze unemployment trends in India using Python and investigate the impact of COVID-19 on employment.

## Dataset

Unemployment in India Dataset

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Analysis Performed

* Data Cleaning
* Exploratory Data Analysis
* Unemployment Trend Analysis
* COVID-19 Impact Analysis
* Regional Comparison
* Correlation Analysis

## Key Findings

* Unemployment increased during the COVID-19 period.
* Significant regional variations exist across India.
* Urban and rural unemployment patterns differ.
* Lockdown periods corresponded to unemployment spikes.

## Policy Recommendations

* Employment generation programs
* MSME support initiatives
* Skill development programs
* Strengthening labour market monitoring
